Restaurant ConFusion

Developing a website for Restaurant ConFusion using HTML, CSS, HTML, JavaScript and Bootstrap. On the home page, a navbar was created with the user in mind; links to the other pages, the restaurant icon, and the login modal for users is housed here. The jumbotron is a carousel that cycles through the highlighted dishes and their descriptions. JavaScript code was used here to dictate the interval legnth and create the pause/play button. The footer is the same on all pages; links to other pages, company info, and buttons for social media shortcuts. The second page shows use of breadcrumbs, cards, accordions and tables showing more detailed information for Restaurant ConFusion. The third page shows the use of forms with a submit button where feedback can be writen and sent to the restaurant. The menu page is being created with React and is currently in production.

JavaScript Matching Game

Manipulating the web page through the DOM structure. Created a basic HTML landing page, then manipulated the 'divs' within the body to display two windows; the left window shows all the smiley faces, the right window is missing one face. To move on to the next level of the game, the user must select the face on left side that is missing on the right. DOM manipulation to randomly place five smiley faces on the left side, then use 'onClick' events to control the progressing of the game or to end the game if the user clicks anywhere else besides the correct smiley face.
